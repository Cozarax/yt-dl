# FetchUrl

Téléchargeur de vidéos simple et rapide. Supporte YouTube, Vimeo, et autres.

## Télécharger l'application

👉 Va dans l'onglet **[Releases](../../releases/latest)** et télécharge `FetchUrl.zip`

## Installation

1. Télécharge `FetchUrl.zip`
2. Extrais le zip où tu veux
3. Double-clique sur `FetchUrl.exe`
4. C'est tout — aucune installation requise

```
FetchUrl/
 ├─ FetchUrl.exe   ← l'application
 └─ bin/           ← ne pas supprimer
```

Le dossier `downloads/` est créé automatiquement au premier téléchargement.

## Utilisation

1. Lance `FetchUrl.exe`
2. Colle le lien d'une vidéo YouTube, Vimeo, etc.
3. Clique sur **Télécharger** ou appuie sur **Entrée**
4. Le fichier `.mp4` apparaît dans le dossier `downloads/`

## Mettre à jour l'application

1. Va dans l'onglet **[Releases](../../releases/latest)**
2. Télécharge le nouveau `FetchUrl.exe`
3. Remplace l'ancien `FetchUrl.exe` par le nouveau
4. Le dossier `bin/` ne change pas, tu le gardes

## Sites supportés

YouTube, Vimeo, Dailymotion, Twitter/X, Instagram, Facebook, et [des centaines d'autres](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md).

## Formats

- Vidéo : MP4, qualité maximale (jusqu'à 1080p)
- Métadonnées et miniature intégrées automatiquement
